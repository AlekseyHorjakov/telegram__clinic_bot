# Telegram Clinic Bot (n8n)

## Описание проекта

Учебный проект по автоматизации медицинского центра с использованием n8n и Telegram.

## Цель проекта

Снизить нагрузку на администраторов за счет автоматизации уведомлений и обработки заявок.

## Реализованный MVP

Workflow:

Manual Trigger
↓
Edit Fields
↓
Telegram Send Message

Система получает данные заявки и отправляет уведомление в Telegram.

## Использованные технологии

* n8n
* Telegram Bot API
* JSON
* Expressions
* JavaScript (базовый уровень)

## Полученные навыки

* Создание workflow в n8n
* Интеграция Telegram
* Работа с JSON
* Expressions в n8n
* Диагностика сетевых проблем

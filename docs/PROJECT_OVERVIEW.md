# Проект: Telegram Clinic Bot

## Бизнес-задача

Администраторы медицинского центра тратят много времени на обработку типовых обращений.

## Решение

Использовать Telegram и n8n для автоматической обработки заявок и уведомлений.

## MVP

1. Получение данных заявки.
2. Формирование сообщения.
3. Отправка уведомления в Telegram.

## Польза для бизнеса

* Снижение нагрузки на сотрудников.
* Быстрое информирование администратора.
* Основа для дальнейшей автоматизации.
